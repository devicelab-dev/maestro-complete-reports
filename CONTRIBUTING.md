# Contributing to Maestro Complete Reports

Thanks for your interest in contributing!

## How to Contribute

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/my-feature`)
3. Make your changes
4. Test with Maestro 2.0.9 or 2.0.10
5. Commit your changes
6. Push to your fork
7. Open a Pull Request

## Reporting Issues

- Check existing issues first
- Include your Maestro version (`maestro --version`)
- Include OS and relevant logs

## Questions?

Open an issue or reach out at [DeviceLab.dev](https://devicelab.dev)
